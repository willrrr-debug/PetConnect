# 快速启动指南

## 一键启动（推荐）

在终端执行：

```bash
cd /Users/will/Downloads/petconnect
./启动.sh
```

第一次运行会提示输入 Supabase Service Key：
1. 访问 https://supabase.com/dashboard
2. Settings > API > 复制 `service_role` key
3. 粘贴到终端

然后浏览器打开：**http://localhost:5173**

## 停止服务

在终端按 `Ctrl+C`

## 如果一键启动失败

### 方法1：手动启动后端
```bash
cd admin-backend
python3 main.py
```

### 方法2：手动启动前端
新开一个终端：
```bash
cd admin-portal
npm run dev
```

然后访问 http://localhost:5173

## 常见问题

**Q: 提示找不到模块？**
运行：`python3 -m pip install --user fastapi uvicorn python-dotenv supabase boto3 python-multipart`

**Q: npm 命令找不到？**
需要先安装 Node.js：https://nodejs.org/

**Q: 图片上传不能用？**
这是正常的，R2 是可选配置。可以使用外部图片 URL。
